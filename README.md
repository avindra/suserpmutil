# suserpmutil

These are various scripts that assist in packaging sources for RPM `.spec`s.

|Script      |Purpose                          |
|------------|---------------------------------|
|vendor-go   |Create Go language vendor tarball|
|vendor-rust |Create Rust vendor tarball       |
|osc-lint    |Expedite spec-cleaner usage      |
